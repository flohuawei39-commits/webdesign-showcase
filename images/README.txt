OPTIONAL PHOTOS
===============

The site is fully self-contained: every visual works out of the box using
CSS gradients and inline SVG art, so nothing here is required.

If you want real photography in two spots, drop image files in THIS folder
with these exact file names and they appear automatically — no code change:

  forest.jpg   -> misty forest / nature photo   (NGO page hero background)
  pitch.jpg    -> football pitch / stadium photo (Sport page club banner)

Recommended: landscape JPGs, ~1600x900, optimised under ~300 KB each for
fast loading. Until the files are added, tasteful colour gradients show in
those spots, so the layout always looks intentional.
